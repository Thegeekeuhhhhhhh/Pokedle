How to Play

Master the game in just a few simple steps

1

Start Guessing

A mysterious Pokémon silhouette appears. You have 6 attempts to guess which Pokémon is hiding behind the shadow.

2

Use the Hints

Each wrong guess reveals detailed hints about the target Pokémon: types, generation, evolution status, and legendary status.

3

Watch the Reveal

With each guess, the shadow becomes more transparent, giving you better visual clues to identify the hidden Pokémon.

4

Win & Score

Guess correctly to earn points based on how few attempts you needed. Build streaks and climb the leaderboard!
